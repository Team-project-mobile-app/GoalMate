# GoalMate

### 1. Ціль застосунку
Створити інтелектуальне середовище, де користувач не просто фіксує свої бажання, а отримує  дорожню карту та менторську підтримку в режимі 24/7 для перетворення абстрактних цілей на конкретні результати.

---

### 2. Які проблеми він вирішує
* **Прокрастинація та відсутність фокусу:** Користувачі часто не знають, з чого почати велику справу.
* **Втрата мотивації:** Без зворотного зв'язку люди кидають справи на півдорозі. AI-чат підтримує, нагадує про цінність мети та допомагає подолати "плато".
* **Інформаційний вакуум:** Коли виникає специфічне питання (наприклад, "яке обладнання купити для блогу?"), AI дає миттєву пораду, позбавляючи потреби годинами шукати в Google.

---

### 3. Типи користувачів (Цільова аудиторія)
* **"Амбітні початківці":** Молоді люди (студенти, фрілансери), які хочуть опанувати нові навички, але потребують структури.
* **"Self-made" підприємці:** Люди, що працюють на себе і потребують асистента для контролю бізнес-цілей.
* **Люди, що дбають про саморозвиток:** Ті, хто ставить цілі у сфері здоров’я, спорту чи навчання (наприклад, "вивчити мову до рівня B2").
* **"Забудькуваті мрійники":** Користувачі, які мають багато ідей, але їм не вистачає дисципліни та дедлайнів.

---

### 4. Основні вимоги до застосунку
* **Створення мети:** Можливість записати будь-яку ціль та встановити для неї кінцевий термін.
* **Інтелектуальний чат:** Доступ до віртуального помічника, який консультує та відповідає на запитання щодо обраної цілі.
* **Покроковий план:** Функція створення списку кроків для досягнення результату.
* **Контроль прогресу:** Можливість відзначати виконані етапи та бачити загальний стан справ.
